
### Objects 
Object-Oriented Programming defines our codes in terms of objects. For example in python we can think of a class as an object. This object can store 

- attributes: pieces of data that belong to the object
- methods: the actions that an object can perform 

 By using object-oriented programming we can write modular code that is robust and intuitive 


### Variables 
Variables are pieces of data which can help describe the current state of an object. 

We define the name of the variable and data type first
```
    float temperature;  
```
then assign a value to it 
```
    temperature= 98.7;  
```

We can also do this together like 
```
    float temperature=98.7;
```
or 
```
    float temperature {98.7}; 
```

### Data Types
Data types refer to the different types of data the variables can take on. The most common are 

- bool: true or false values
- int: integer
- float: decimal floating number
- char: individual characters 
- string: a collection of characters
